Tạo trang Web với html

  1. HTML (HyperText Markup Language) là tập hợp các thẻ và quy ước được thiết kế để tạo nên các trang web và hiển thị bởi trình duyệt. HTML cho phép người tạo nội dung kết hợp văn bản với hình ảnh, âm thanh, video và các liên kết đến những trang khác. Mặc dù HTML rất quan trọng trong xây dựng giao diện web, nó không phải là một ngôn ngữ lập trình (không có logic tính toán giống như JavaScript hay ngôn ngữ phía server).
